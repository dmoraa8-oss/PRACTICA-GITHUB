# ⏱️ Cronómetro - Práctica Git/GitHub

App web simple hecha en HTML/CSS/JS para demostrar el flujo de trabajo 
colaborativo con Git y GitHub.

## 🚀 Demo
Abre `index.html` en cualquier navegador.

## 👥 Equipo
- Persona A — Base (start/pause/reset)
- Persona B — Estilos
- Persona C — Vueltas (laps)
- Persona D — Récord (localStorage)

## 🛠️ Features
- ✅ Cronómetro con formato HH:MM:SS
- ✅ Botones de iniciar, pausar y reiniciar
- ✅ Registrar tiempos por vuelta
- ✅ Guardar mejor tiempo
